# ZRenderForMobile
一个对ZRender移动端的扩展插件，支持在h5端在canvas上进行手势识别，包括旋转和缩放等操作

## 关于ZRender 
ZRender是echarts的2d渲染库
https://github.com/ecomfe/zrender
